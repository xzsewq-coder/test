# 홍보용 개인 블로그 설계 문서

**날짜:** 2026-04-08  
**목적:** 강의 및 홈페이지 제작 의뢰 유치를 위한 개인 홍보 블로그

---

## 개요

HTML/CSS/JS 기반 정적 사이트. GitHub Pages로 호스팅. 단일 페이지 랜딩 구조에 블로그 섹션 포함. 강의 신청과 홈페이지 제작 의뢰 두 가지 CTA를 각각 운영.

---

## 파일 구조

```
프로젝트 루트/
├── index.html              # 메인 랜딩 (모든 섹션 포함)
├── style.css               # 전체 스타일
├── main.js                 # 인터랙션 (스크롤, 네비게이션)
├── posts/
│   ├── index.html          # 블로그 목록 페이지
│   └── YYYY-MM-DD-제목.html # 개별 블로그 글
└── assets/
    ├── images/             # 포트폴리오 스크린샷, 프로필 사진
    └── fonts/              # 로컬 폰트 (선택)
```

---

## 메인 페이지 섹션 구성

| 순서 | 섹션 | 내용 |
|------|------|------|
| 1 | Hero | 이름 + 한 줄 소개 + CTA 버튼 2개 (강의 신청 / 제작 문의) |
| 2 | About | 자기소개, 경력, 전문성 |
| 3 | Services | 강의 서비스 카드 + 홈페이지 제작 카드 |
| 4 | Portfolio | 제작 작업물 그리드 (스크린샷 + 외부 링크) |
| 5 | Blog Preview | 최근 글 3개 카드 → "전체 보기" 링크 |
| 6 | Contact/CTA | 강의/제작 각각 연락처 버튼 |
| 7 | Footer | 링크, 저작권 |

---

## 디자인 시스템

- **배경:** `#0d1117` 계열 다크
- **포인트 컬러:** 네이비/블루 계열
- **텍스트:** 흰색 계열
- **폰트:** Pretendard (한국어 최적화, Google Fonts 또는 CDN)
- **톤:** 전문적이고 신뢰감 있는 느낌

---

## 인터랙션

- **네비게이션:** 상단 고정 헤더, 스크롤 시 배경 불투명 전환, 스무스 스크롤
- **모바일:** 햄버거 메뉴, 반응형 레이아웃
- **포트폴리오:** 3열 그리드 (모바일 1열), 호버 시 오버레이 (프로젝트명 + "보기" 버튼)

---

## CTA 동작

| CTA | 동작 |
|-----|------|
| 강의 신청 | 오픈채팅 또는 신청 폼 URL (`href` 링크) |
| 제작 문의 | 이메일 (`mailto:`) 또는 카카오톡 링크 |

---

## 블로그 관리 방식

새 글 작성 시:
1. `posts/YYYY-MM-DD-제목.html` 파일 생성
2. `index.html`의 Blog Preview 섹션에 카드 1개 추가 (최신 3개 유지)
3. `posts/index.html` 목록에 링크 추가
4. 소스트리에서 커밋 & 푸시 → GitHub Pages 자동 배포

---

## 배포

- **호스팅:** GitHub Pages (`main` 브랜치 자동 배포)
- **Git 워크플로우:** 소스트리(SourceTree)로 커밋/푸시 직접 처리
- **커스텀 도메인:** 필요 시 `CNAME` 파일 추가
